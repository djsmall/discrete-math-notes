Boolean Algebra
...

Summary

Variables that have a value of 1 or 0 are Boolean variables.
In Boolean algebra, variables, expressions and evaluations can only be 0 (False) or 1 (True).
The main boolean operations are the following, with their symbolic representation:

  • Multiplication (Conjunction): AND ∧
  • Addition (Disjunction): OR ∨
  • Complementary (Negation): NOT ¬

Precedence Rules:

  • Multiplication takes precedence over addition
  • Complement is applied when the expression under the bar is evaluated
  • Parenthesis overrides any other precedence
  • When evaluating expressions, it's often a good idea to place complement expressions in parenthesis so the order of operations is clear (Except is it's applied for a single variable or the whole expression)

Simplified Order of Operations:

  1. ( ) parenthesis
  2. ¬ (not)
  3. ∧ (and)
  4. ∨ (or)

Logic and proposition
  • Logic is the study of formal reasoning. A statement in logic always has a well defined meaning.
  • Proposition is a statement that is either true or false, and is the most basic element of logic.

The Truth Value in a proposition is the value that a proposition has, indicating whether it is true or false. While a proposition itself can be true in the sense that it is a valid proposition, it's value may be either true or false. e.g. "2 + 2 = 5" is as true proposition, but the value itself is false. Note that a truth value of a valid proposition can also be unknown, or subjective ("It's raining outside" can be true for some people and false for others).

Logical operations are similar to boolean operations, in they take the form ∧ (AND), ∨ (OR), or ¬ (NOT).

Compound proposition
...

A Compound proposition is one that connects singular propositions with logical operations, similar to ∧ (AND).

Example

The proposition p ∧ q can be expressed in plain English as:
January has 31 days and February has 33 days.
(This is a true proposition, but the truth value of it is false, February does not have 33 days.)

Disjunction operation
...

A Disjunction operation is the equivalent of the ∨(OR) boolean operator. Just one proposition in the statement needs to be true for the evaluation to be true, only when they are both false is the entire statement to be considered false. This is also known as the inclusive or to distinguish it from the exclusive or.

Example

The proposition p ∨ q can be expressed in plain English as:
January has 31 days or February has 33 days.
(This is a true proposition, and the truth value of it is true, because January does have 31 days, so the proposition of February having 33 days is irrelevant.)

Exclusive Or
...

Exclusive Or is another type of or statement, but for situations where the proposition statement is exlusively true, and evaluates to false if both are true (and false if both are false).

Example

The proposition p ⊕ q can be expressed in plain English as:
"Lucy is going to the park or the movie"
This is distinguished from the inclusive or because the wording implies she is doing one over the other, and can't do both at the same time

Conditional operation

The proposition p → q is read in plain English as:
"if p then q".
A compound proposition that uses this operation is a conditional proposition.
The proposition p is the hypothesis, whereas the proposition q is the conclusion.
The only way for a conditional statement to be false is if the hypothesis is true and the conclusion is false.

Three types of Conditional Operations:

Propositionp → q"If it is raining, the game will be cancelled"
Converse:q → p"If the game is cancelled, it is raining"
Contrapositive: ¬q → ¬p"If the game is not cancelled, then it is not raining"
Inverse: ¬p → ¬q"If it is not raining, the game will not be cancelled"

Note: The converse and the inverse are logically equivalent, but they are not logically equivalent to the original proposition.
